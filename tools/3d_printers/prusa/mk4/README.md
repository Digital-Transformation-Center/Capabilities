# Original Prusa MK4

The Original Prusa MK4 is an open cartesian (bed-slinger) 3D printer. 

## Getting Started

If you have not used the Prusa MK4 before, or otherwise need help with 3D printing, contact [Ryan Kuederle](mailto:kuederler1@udayton.edu).

### Preparing Your Print

To print a model with the Prusa MK4, it is recommended to use Prusa's [Prusa Slicer](https://www.prusa3d.com/p/prusaslicer/). 

The DTC's available Prusa MK4 should be selected in the Prusa Slicer as the *Original Prusa MK4 Input Shaper*, with a 0.4mm nozzle.
