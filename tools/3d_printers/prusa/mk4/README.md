# Original Prusa MK4

The Original Prusa MK4 is an open cartesian (bed-slinger) 3D printer. 

## Getting Started

If you have not used the Prusa MK4 before, or otherwise need help with 3D printing, contact [Ryan Kuederle](mailto:kuederler1@udayton.edu).

### Preparing Your Print

To print a model with the Prusa MK4, it is recommended to use Prusa's [Prusa Slicer](https://www.prusa3d.com/p/prusaslicer/). 

The DTC's available Prusa MK4 should be selected in the Prusa Slicer as the *Original Prusa MK4 Input Shaper*, with a 0.4mm nozzle. 

![PrusaSlicer printer selection](images/printer_selection.png)

Within Prusa Slicer, be sure to select a filament that matches the installed filament. Typically, the "Generic" filaments are adequate.

**If you have not printed with Prusa Slicer before**, visit Prusa's [First Print with PrusaSlicer](https://help.prusa3d.com/article/first-print-with-prusaslicer_1753?product=prusaslicer) article.

### Starting a Print

Before starting a print, ensure the build plate is clear and properly installed. The spring steel build plate is held magnetically to the machine, and can be removed by grabbing at any of the corners. At the top of the heatbed are two pins that align perfectly with the notch in the sheet. Ensure the sheet is fully aligned and flat on the surface before starting a print. 

![Bed alignment pin](images/alignment_pin.png)

A USB storage drive can be used to upload files to the MK4. After uploading your G-code, insert the USB stick into the USB port on the side of the MK4's display, then the recently uploaded model will be displayed. If the desired model is not present, use the control knob (turn to move, press to select) to navigate to the appropriate model, and select start. 

![Screen](images/prusa_screen.jpeg)

### After Your Print

When the print is complete, the spring steel sheet can be removed from the build surface. Do not touch the surface of the build plate, as fingerprint residue can prevent future prints from properly adhering. 

Whlie grasping the edges of the steel plate, gently bend it to help release parts. Use a plastic scraper to remove any additional material on the surface.
